# How to Contribute

Required: Git, Maven.

```
git clone https://github.com/metricminer-msr/metricminer2.git
cd metricminer2
```

Then, you can:

compile : `mvn clean compile`
test    : `mvn test`
eclipse : `mvn eclipse:eclipse`
build   : `mvn clean assembly:single`
