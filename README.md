# How to Contribute

Required: Git, Maven.

```
git clone https://github.com/mauricioaniche/metricminer2.git
```

Then, you can:

* compile : `mvn clean compile`
* test    : `mvn test`
* eclipse : `mvn eclipse:eclipse`
* build   : `mvn clean compile assembly:single`
